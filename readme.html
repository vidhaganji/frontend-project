Country Info Web Application Readme

This README file provides an overview of how to create a web application that displays information about a country using the RestCountries API, Python Flask, HTML, and CSS. The web application will allow users to search for a country by its name or code and view various details about the country.

Prerequisites

Before you start, make sure you have the following tools and libraries installed:

  • Python: You will need Python to run the Flask application. You can download it from python.org.
  • Flask: Flask is a lightweight web framework for Python. You can install it using pip: 
    pip install flask
  • HTML and CSS: Basic knowledge of HTML and CSS is required for creating and styling web pages.
  • Text editor or IDE: You can use any code editor or integrated development environment of your choice (e.g., Visual Studio Code, Sublime Text, PyCharm).

Set Up Your Project Directory

Create a new directory for your project and navigate to it in your command line or terminal.

HTML Templates

Create two HTML templates in a "templates" folder in your project directory: index.html and country_info.html. The index.html template will contain the search form, and the country_info.html template will display information about the selected country.

CSS Styles

Create a CSS file (e.g., styles.css) to style your web pages and include it in your HTML templates.
